# Assignment Knowledge Base

你可以把这里的内容当做《手册的手册》，或者说，像是《南瓜书》之于《西瓜书》的关系。

我们建议您充分阅读了实验手册之后，遇到了一些不太明白的地方，再来阅览本仓库。

## 内容介绍

在《手册的手册》中，包含但不限于以下内容：

* 哪些文件需要你从上一次实验中复制过来……
* 在某个方法的实现上，有哪些同学们经常问到的问题……
* 对于没有给出算法的内容，从哪里起手……
